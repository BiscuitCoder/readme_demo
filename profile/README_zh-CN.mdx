## 🚀 主线构建路径

BuildPath是当前研发 DAO 运行的主线，通过下列的的核心项目，形成从人才培养到项目落地，再到最后的激励循环，将有良心的开发者不断汇聚进来建设，使得有源源不断的Web3新技术、应用的可以研发落地。同时产出全部是开源项目，过程和资金全部公开透明，以此缓解开源项目 人 + 钱 的可持续性问题。

| 链接 | 描述 | 进度 |
|------------|-------------|------|
| <a href="https://github.com/IntensiveCoLearning" target="_blank">📚 IntensiveCoLearning</a> | 让更多人学习需要用到的技术 | `50%` |
| <a href="https://github.com/lxdao-official/WhatToBuild" target="_blank">💡 WhatToBuild</a> | 分析、研究、整理创建应用想法，给大家创作灵感，避免重复做基础轮子 | `10%` |

## 🏗️ 关于Pod
Pod是 LXDAO 快速、轻量、自主自发的工作小组，它成立无需社区提案，适合早期项目、Research，以及任何小型项目。旨在为有潜力的项目提供资金、资源和指导，帮助它们成长为成熟的项目。

任何 LXDAO 成员在提出Pod申请后，将由 LXDAO 工作组成员公开评审并投票决定是否成立。

通过申请 Pod 你将获得来自社区的资金、开发、运营等资源支持。

- [✍️ 申请流程](https://docs.lxdao.io/docs/zu-zhi-jia-gou/pod)
