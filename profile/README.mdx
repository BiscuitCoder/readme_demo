## 📚 About LXDAO




## 🚀 BuildPath

BuildPath is the main line of the current R&D DAO operation. Through the following core projects, it forms a cycle from talent cultivation to project implementation, and finally to incentive circulation. This will continuously gather conscientious developers to build, enabling a continuous flow of Web3 new technologies and applications to be developed and implemented. All outputs are open source projects, with the process and funds fully transparent, thereby alleviating the sustainability issues of open source projects in terms of people + money.

| Project | Description | Progress |
|------------|-------------|------|
| <a href="https://github.com/IntensiveCoLearning" target="_blank">📚 IntensiveCoLearning</a> | Let more people learn the technologies they need | `50%` |
| <a href="https://github.com/lxdao-official/WhatToBuild" target="_blank">💡 WhatToBuild</a> | Analyze, research, and organize application ideas to inspire everyone's creativity and avoid reinventing basic wheels | `10%` |
| <a href="https://github.com/Casual-Hackathon" target="_blank">🎮 CasualHackathon</a> | Casual hackathon, no prize pool, no PR, returning to pure hackathon, letting everyone brainstorm and quickly create demos and MVPs [learn more 👉](https://forum.lxdao.io/t/the-introduction-of-casual-hackathon/2827) | `10%` |
| <a href="https://fairsharing.xyz" target="_blank">🤝 FairSharing</a> | Record contributions and collaborate together, allowing everyone to work asynchronously, distribute fairly, and contribute persistently | `70%` |
| <a href="#" target="_blank">🏆 RetroFunding</a> | Retroactive rewards, selecting projects that work seriously and produce results, adding additional and continuous incentives | `0%` |




## 🏗️ About Pod
Pod is a fast, lightweight, and autonomous working group in LXDAO. It can be established without community proposals, suitable for early-stage projects, Research, and any small projects. The establishment of Pod aims to provide funding, resources, and guidance to promising projects, helping them grow into mature projects.

Any LXDAO member can apply for a Pod, which will be publicly reviewed and voted on by the LXDAO working group to decide whether to establish it.

[Current Pod Groups Overview 👉](https://www.notion.so/lxdao/Pod-1eadceffe40b804ea6fdf7a29f4edd4c)
